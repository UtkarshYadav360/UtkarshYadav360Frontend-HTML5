"th element" is used to define heading


"border = "" attribute" is used to set the border around the table.

"colspan = "" attribute" is used to combine columns

"rowspan = "" attribute" is used to combine rows

"caption element" is used to specify the table name or the information about the data that table contains

"thead element" is used to define the table header or the heading of the table.

"tbody element" contains the data of the table.

"tfoot element" is same as "thead element", but it helps when the table is scrollable and too big.It can also contain the headings of the table.
