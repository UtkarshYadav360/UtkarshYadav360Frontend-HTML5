Forms are used to take user input


  1. "form element" is used to make a form.

  2. "input tag" is used to take input from the user.

  3. "type = "" attribute" is used to specify the input type.

  4. type = "submit" is used to submit the form.Text inside the value = "" attribute will be displayed inside the submit button.


Form Input Types :

  1. type = "text"(text type input)

  2. type = "submit"(submit button)

  3. type = "color"(color selector)

  4. type = "date"(calendar to choose date, day and month)

  5. type = "email"(email input)

  6. type = "number"(only number input)

  7. type = "password"(password input)

  8. type = "reset"(reset the form)

  9. type = "time"(specify the time input, for hour, minute and AM or PM)

  10. type = "url"(only url as input)

  11. type = "range" min = "minimum price" max = "maximum price" (only for specific range of numbers mostly used in e-commerce websites)

  12. TYPE enctype = "multipart/form-data" INSIDE OPENING FORM TAG
    TYPE type = "file" INSIDE INPUT TAG
    "multiple attribute" can be placed inside the input tag to select multiple files.
