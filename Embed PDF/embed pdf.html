"embed element", "iframe element" and "object element" can be used to embed pdf


Using Embed Element :

"src = "" attribute" is used to tell the path of the pdf.

"type = "application/pdf" attribute" can be used to tell the type of the pdf.

width and height can be specified.


Using Iframe Element :

"src = "" attribute" is used to tell the path of the pdf.

no need to use type = "" attribute.

width and height can be specified.


Using Object Element :

"data = "" attribute" is used to tell the path of the pdf.

"type = "application/pdf" attribute" can be used to tell the type of the pdf.

width and height can be specified.
