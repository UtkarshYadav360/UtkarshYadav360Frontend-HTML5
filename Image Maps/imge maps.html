Image Maps is a technique, which is used when we want to use more than one link on an image


STEPS TO USE IMAGE MAPS :

  1. Use "img tag" for the selected image to have multiple links on it.

  2. put "usemap = "" attribute" in the "img element".It takes an id as it's value.

  3. Make a "map tag".It is used to define an image map.Image map is an image with clickable areas.

  4. Put name = "" attribute" in the "map tag". It should have the same value as "usemap attribute" have.

  5. Make an "area tag".It defines the area in the image map which will be clickable.

  6. Put "shape = "" attribute" and "coords = "" attribute" in the area tag.

  7. shape attribute can have default, rect, poly, circle as it's value.

  8. "coords attribute" have the coordinates of that part of the image, where we have to put the link. Number of coordinates depends on the shape we choose in shape attribute.

SHORTCUT :

Use imagemap.org or a similar website to automatically generate the code when we drag our image and set a link to it
